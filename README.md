# Task Management System

A full-stack Task Management web application built with React and Spring Boot.

## Tech Stack
- **Frontend:** React, Tailwind CSS, Vite
- **Backend:** Spring Boot, Java, Spring Security, JWT
- **Database:** MySQL

## Features
- User Authentication (Register/Login with JWT)
- Create and manage Projects
- Create, assign and track Tasks
- Dashboard overview
- Role-based access

## How to Run

### Backend
```bash
cd taskflow/backend
mvn spring-boot:run
```

### Frontend
```bash
cd taskflow/frontend
npm install
npm run dev
```
